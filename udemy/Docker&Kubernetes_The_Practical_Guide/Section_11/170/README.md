# 170. 수동 배포의 더 많은 문제점들

Kubernetes (k8s)가 컨테이너화된 애플리케이션의 배포, 확장 및 관리를 자동화하기 위한 오픈 소스 시스템이다. 즉 컨테이너 배포에 도움이 되는 것, 도구 및 작업 방법의 모음이다.

컨테이너를 수동으로 배포하게 되면 EC2 인스턴스를 자체적으로 관리하고 구성해야하며 유지보수하기 힘들다. 
- 컨테이너가 충돌하거나 다운될 수 있으며 새 컨테이너로 교체해야한다. 
- 트래픽의 증감에 따라 컨테이너의 인스턴스를 늘리거나 줄여야해서 관리가 어렵다.
- 트래픽이 들어왔을 때 이를 균등하게 분산되게 해야한다.
